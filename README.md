# Sistema de Gestión de Tareas

Este proyecto corresponde a un Sistema de Gestión de Tareas desarrollado con Next.js y React 19, que incluye autenticación simulada mediante JWT, gestión de estado y uso de mocks para simular una API.

## Instalar dependencias

```bash
npm install
# or
yarn install
```

## Iniciar el Proyecto

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Al abrir [http://localhost:3000](http://localhost:3000) podrá ver el resultado.

## Ejecutar Tests

```bash
npm run test
```
